# ⚛️ AtomVerse

> **Explore the Periodic Table Like Never Before**

AtomVerse is a modern, interactive, and visually immersive periodic table built with **React, TypeScript, Vite, and Tailwind CSS**. Designed with a premium dark interface, fluid animations, and detailed scientific information, it transforms the traditional periodic table into an engaging learning experience.

Whether you're a student, educator, chemistry enthusiast, or simply curious about the elements that make up our universe, AtomVerse offers an intuitive way to explore every known element—from Hydrogen to the predicted Element 119.

## ✨ Features

### 🎨 Modern User Interface

* Premium dark theme with vibrant category-based colors
* Fully responsive layout for desktop and mobile devices
* Smooth animations and transitions
* Interactive hover effects and micro-interactions
* Glassmorphism-inspired design language

### ⚛️ Interactive Periodic Table

* Complete periodic table with all **118 discovered elements**

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/FireStormy1/atomverse.git
```

### Navigate into the project

```bash
cd atomverse
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open:

```
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
